TwitterCounter4J
================
The below mentioned libraries are required for compiling the project.

Apache HttpClient
-----------------
commons-codec-1.6.jar
commons-logging-1.1.3.jar
fluent-hc-4.3.3.jar
httpclient-4.3.3.jar
httpclient-cache-4.3.3.jar
httpcore-4.3.2.jar
httpmime-4.3.3.jar

Google GSON
-----------
gson-2.2.4.jar
